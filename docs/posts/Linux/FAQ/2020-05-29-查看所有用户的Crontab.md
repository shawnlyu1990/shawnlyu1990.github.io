root身份查看 `/var/spool/cron/` 目录下的文件即可

```bash
[root@localhost ~]# cd /var/spool/cron/
[root@localhost cron]# ll
总用量 16
-rw-------. 1 lisi        lisi         43 6月  21 2016  lisi
-rw-------  1 root        root        703 1月   2 18:35 root
-rw-------. 1 zhangsan    zhangsan    619 6月  21 2016  zhangsan
```
