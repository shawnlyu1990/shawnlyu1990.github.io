【FAQ】Linux 常用命令（以 CentOS 为例）

原创昌霖学长LinuxFAQFAQLinuxCentOS常用命令大约 2 分钟约 610 字
Bash 命令补全增强软件包
yum install -y bash-completion bash-completion-extras
镜像一个网站到本地（扒站）

该命令可以用来镜像一个网站，wget将对链接进行转换。如果网站中的图像是存放在另外的站点，那么可以使用 -H 选项。

cd /var/www/html/
wget -m -k -H http://www.example.com/
CentOS 7 安装和使用 iptables

安装 iptables

# 安装iptables
yum install -y iptables

# 安装iptables-services
yum install -y iptables-services

iptables 实现 SNAT

# MASQUERADE 方式
/sbin/iptables -t nat -A POSTROUTING -o ppp0 -j MASQUERADE

iptables 保存配置

# 保存iptables配置
iptables-save > /etc/sysconfig/iptables
CentOS 7 安装和使用 bridge

安装 bridge-utils

yum install -y bridge-utils

编辑 /etc/sysconfig/network-scripts/ifcfg-br0

TYPE=Bridge
BOOTPROTO=static
DEFROUTE=yes
PEERDNS=yes
PEERROUTES=yes
NAME=br0
DEVICE=br0
ONBOOT=yes
IPADDR=192.168.1.1
NETMASK=255.255.255.0

编辑 /etc/sysconfig/network-scripts/ifcfg-eth1

TYPE=Ethernet
BOOTPROTO=none
DEFROUTE=yes
PEERDNS=yes
PEERROUTES=yes
NAME=eth1
DEVICE=eth1
ONBOOT=yes
BRIDGE=br0   # 新增

重启网络服务

systemctl restart network
CentOS 7 添加静态路由

编辑路由文件 /etc/sysconfig/network-scripts/route-eth0

# 目的网段/掩码 via 下一跳 dev 出接口 

172.16.0.0/24 via 172.24.18.2 dev eth0

重启网络服务

systemctl restart network
CentOS 7 开启路由转发
# 临时开启
echo "1" > /proc/sys/net/ipv4/ip_forward

# 永久开启
echo "net.ipv4.ip_forward = 1" >> /etc/sysctl.d/99-sysctl.conf

# 重载规则
sysctl -p
